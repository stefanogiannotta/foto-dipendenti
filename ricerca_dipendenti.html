⚙️ Configurazione necessaria (una volta sola)

Per far funzionare l'app, apri questo file HTML con un editor di testo (es. Blocco Note) e modifica le due righe nella sezione CONFIGURAZIONE:

  1. API_KEY → la chiave che otterrai da Google Cloud Console (vedi guida sotto)
  2. FOLDER_ID → l'ID della cartella Drive con le foto (è nell'URL della cartella)

Come ottenere l'API Key in 5 minuti:

  1. Vai su console.cloud.google.com con l'account Google aziendale
  2. Crea un nuovo progetto (nome es: "Foto Dipendenti")
  3. Menu laterale → API e servizi → Libreria → cerca Google Drive API → Attiva
  4. Menu laterale → Credenziali → Crea credenziali → Chiave API
  5. Copia la chiave e incollala nel file al posto di LA_TUA_API_KEY_QUI
  6. (Facoltativo ma consigliato) Clicca sulla chiave → Limitazioni applicazione → Siti web HTTP → aggiungi il dominio da cui la aprirete

Come trovare l'ID della cartella Drive:

  1. Apri la cartella delle foto su Google Drive
  2. Copia l'URL dalla barra del browser, es: https://drive.google.com/drive/folders/1aBcDe...
  3. La parte dopo /folders/ è l'ID → incollala al posto di 1euOZuc_lRRVPis-6f22dnVo5tdj-rQd6
